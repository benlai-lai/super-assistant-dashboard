# 超級隨身助理 — 第一版產品規格

## 1. 產品定位
一個以「降低認知負擔」為核心的個人工作中控台。
使用者以自然語言輸入想法、任務與行程；系統將內容整理成結構化資訊，並以一眼可判斷的 Dashboard 呈現。

## 2. 核心原則
1. 使用者不必學習系統，系統應配合使用者的工作方式。
2. 工作狀態優先，日曆只是其中一個資訊區塊。
3. 先提供摘要與判斷入口，需要時才往下查看細節。
4. 任務、行程與想法分流，避免所有內容都變成待辦。
5. 桌機與手機皆可使用，採響應式網頁設計。

## 3. 第一版範圍
### 包含
- 總覽首頁
- 今日任務
- 已完成任務
- 進行中任務
- 逾期任務
- 今日行程時間軸
- 本週任務狀態
- 專案進度概覽
- 即將到期事項
- 小型月曆
- 快速新增入口
- 桌機版與手機版響應式介面
- 可點擊導覽與基本互動

### 暫不包含
- Google Calendar / Apple Calendar 同步
- 真正的語音辨識
- 後端資料庫
- 多人協作
- 帳號登入
- 自動提醒推播
- Codex 或其他 Agent 的實際串接

## 4. 資料類型
### 想法
尚未承諾執行，只存入 Inbox。

### 任務
沒有固定發生時間，但有狀態、優先級與完成條件。

### 行程
具有明確日期與時間，時間經過後標記為已結束。

### 專案
由多個任務組成，顯示整體進度、下一步與風險。

## 5. 狀態規則
- 未開始
- 進行中
- 等待決策
- 已完成
- 逾期
- 已封存

任務需要由使用者或系統確認完成。
行程在時間經過後標記為已結束，但保留在歷史紀錄中。

## 6. 第一版首頁資訊優先順序
1. 今天最重要的三件事
2. 今天的行程
3. 等待使用者決策
4. 進行中的專案
5. 即將到期與逾期事項
6. 任務整體狀態
7. 月曆

## 7. 成功標準
- 使用者在 30 秒內理解今天最重要的工作狀態。
- 手機上可以快速查看任務、行程與專案。
- 使用者不需要翻閱多份 Markdown 才知道目前進度。
- 新增一件事不超過一個主要動作。
